我们是通过 git clone命令来将该仓库下载到本地的，这个时候的remote其实是原项目地址，我们当然是不能直接git push的，因为我们并没有这样的权限
建立远程连接
git remote add origin https://github.com/jjc123/one.git
取消本地目录下关联的远程库：
git remote remove origin
